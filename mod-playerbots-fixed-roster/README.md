# mod-playerbots-fixed-roster

Small AzerothCore module for fixed WSG filler bots on the 19 PvP server.

Install by linking this directory into the core modules directory before configuring/building:

```bash
ln -s /root/services/19pvp/mod-playerbots-fixed-roster /root/services/19pvp/core/modules/mod-playerbots-fixed-roster
```

The module reads `PlayerbotsDatabase.playerbots_fixed_roster` and logs in enabled character GUIDs through `sRandomPlayerbotMgr.AddPlayerBot(...)`. It does not patch `mod-playerbots` and does not require `AiPlayerbot.RandomBotAutologin`.

Roster data is generated by `tasks/refresh_db.ts` from the `WSG_BOT` Google Sheet tab into `core_scripts/generated-playerbots-fixed-roster.sql`.

After applying roster SQL, reload without restarting:

```text
.wsgbots reload
```
